# Zasady dla Cursor w tym repozytorium

- Pisz opisy commitów wyłącznie po polsku.
- Format commitów:
  - Pierwsza linia: tryb rozkazujący, maks. 72 znaki, bez kropki
  - Pusta linia
  - Krótkie punkty opisujące zmiany
- Używaj polskich nazw i sformułowań; unikaj zbędnych anglicyzmów.
- Prefiksy (Conventional Commits po polsku):
  - feat: Nowa funkcja
  - fix: Poprawka błędu
  - docs: Dokumentacja
  - refactor: Refaktoryzacja
  - perf: Wydajność
  - chore: Porządki/konfiguracja

Przykład:

feat: Dodaj statystyki grup z kolorami wg zajętości

- Liczenie członków przez pivot members()
- Kolory i tooltipy zależne od limitu i zajętości
- Karta „Grupy pełne” i „Grupy z miejscem”
