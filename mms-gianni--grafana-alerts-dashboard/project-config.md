# Grafana Alerts Dashboard - Copilot Instructions

## Project Overview
Alternative dashboard for Grafana alerts optimized for wall screen displays.

## Tech Stack
- **Backend**: NestJS with WebSocket Gateway
- **Frontend**: Vue 3 + Vite + PrimeVue + tailwindcss 4
- **Real-time Updates**: Socket.io connections

## Development Guidelines
- Use TypeScript for all code
- Follow NestJS best practices for backend services
- Use Vue 3 Composition API
- Leverage PrimeVue components for UI
- Implement real-time updates via WebSockets

## Project Structure
- `/backend` - NestJS application
- `/frontend` - Vue 3 + Vite application
- Root workspace with shared scripts
