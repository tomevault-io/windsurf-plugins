## Project Structure
```
src/
├── app/
│   ├── globals.css     # Global styles and custom CSS
│   ├── layout.tsx      # Root layout component
│   └── page.tsx        # Main page component
└── components/
    ├── Navbar.tsx      # Navigation component
    ├── Hero.tsx        # Hero section with typing animation
    ├── About.tsx       # About section
    ├── Education.tsx   # Education timeline
    ├── Skills.tsx      # Technical skills with progress bars
    ├── Projects.tsx    # Project showcase with filtering
    ├── Research.tsx    # Research and publications
    ├── Contact.tsx     # Contact form and info
    └── Footer.tsx      # Site footer
```

## Coding Standards
- Use TypeScript for all components

## Component Guidelines
- Each component should be self-contained in its own file

- Use Tailwind CSS custom theme colors (primary/secondary palettes)
