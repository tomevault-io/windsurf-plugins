# LongevAI — Project Instructions

## No Emojis — Mandatory Rule

**NEVER use emoji characters anywhere in this codebase.**

This includes components, pages, data files, labels, badges, alerts, buttons, and any UI element.

Instead, use:
- **Lucide React icons** (already installed — `lucide-react`)
- **SVG icons** (inline or as components)
- **Other icon libraries** if needed (e.g., Heroicons, Phosphor)
- **CSS-based graphics** or any other form of digital art

Any existing emojis in the code should be replaced with proper icon components when touched.
