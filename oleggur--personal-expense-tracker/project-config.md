# Personal Expense Tracker - AI Development Documentation

## Project Overview

A **fully functional web application** for tracking personal income and expenses. Users can register accounts, create custom categories, record transactions, view statistics, and export their data to CSV.

**Core Value**: Simple financial tracking without complexity of full accounting software or limitations of spreadsheets.

**Required Features**:
- User authentication (register, login, logout)
- Category management (create, view, delete income/expense categories)
- Transaction management (add, view, delete transactions with filters)
- Statistics dashboard (spending by category, time period analysis)
- CSV export functionality
- Multi-user support with data isolation
