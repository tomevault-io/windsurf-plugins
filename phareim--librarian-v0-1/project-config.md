# Librarian v0.1

Read-it-later app (open-source Pocket alternative). Save articles via web UI, browser extension, or Android share sheet.

## Tech Stack

- Next.js 14, React 18, TypeScript, Tailwind CSS
- Firebase Cloud Functions (Node 20), Cloud Firestore
- Mozilla Readability for article parsing
- OpenAI GPT-3.5 for AI summaries

## Commands

- `pnpm i` — install dependencies
- `firebase login && firebase use --add` — set up Firebase
- `pnpm dev` — dev server + Firebase emulators
