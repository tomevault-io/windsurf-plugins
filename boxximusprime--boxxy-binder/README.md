# Boxxy-Binder

> Tome by [BoxximusPrime](https://github.com/BoxximusPrime/Boxxy-Binder) — distributed by [TomeVault](https://tomevault.io)

A desktop application for managing joystick, keyboard, and mouse key bindings for Star Citizen outside of the game. Built with Tauri 2.0 and Rust, this tool provides a powerful interface to organize, customize, and debug your control scheme.

## Windsurf Config

The `project-config.md` file in this directory is the project config converted for Windsurf.
Original source: `copilot-instructions.md` in [BoxximusPrime/Boxxy-Binder](https://github.com/BoxximusPrime/Boxxy-Binder).
