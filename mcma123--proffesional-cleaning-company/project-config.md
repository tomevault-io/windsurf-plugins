# Project Overview

This is a React-based web application for a professional cleaning company. It is built with Vite, TypeScript, and styled with Tailwind CSS and shadcn/ui. The application features a public-facing website with pages for home, about, services, gallery, shop, and contact. It also includes an admin section.

## Technologies Used

*   **Frontend:** React, Vite, TypeScript, Tailwind CSS, shadcn/ui
*   **Routing:** React Router DOM
*   **Data Fetching:** Tanstack Query
*   **Forms:** React Hook Form
*   **UI Components:** shadcn/ui, Radix UI

## Project Structure

*   `src/` - Contains the main source code for the application.
    *   `components/` - Reusable UI components.
    *   `pages/` - Top-level page components for each route.
    *   `App.tsx` - The root component that sets up routing.
    *   `main.tsx` - The entry point of the application.
*   `public/` - Contains static assets like images and videos.
*   `package.json` - Defines the project's dependencies and scripts.
*   `vite.config.ts` - Configuration file for Vite.
*   `tailwind.config.ts` - Configuration file for Tailwind CSS.

## Building and Running

To get the project up and running, follow these steps:

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```
    This will start the development server at `http://localhost:8080`.

3.  **Build for production:**
    ```bash
    npm run build
    ```
    This will create a `dist` directory with the production-ready files.

## Development Conventions

*   The project uses TypeScript for static typing.
*   Styling is done with Tailwind CSS and shadcn/ui.
*   Components are organized by feature in the `src/components` directory.
*   Pages are located in the `src/pages` directory.
*   The project uses ESLint for linting.
