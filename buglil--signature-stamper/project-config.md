## Project Overview

This is a React TypeScript application built with Vite for PDF preview and signature stamping functionality. The application allows users to:

- Load and preview PDF documents
- Add signature stamps to specific locations on PDF pages
- Save modified PDF documents

## Key Technologies

- **React 18** with TypeScript
- **Vite** for build tooling
- **react-pdf** for PDF rendering and display
- **pdf-lib** for PDF manipulation and signature injection
- **react-signature-canvas** for signature creation
- **react-dropzone** for file upload
- **lucide-react** for modern icons
