# Vue 3 + TypeScript + Vue-Flow 流程编排系统

这是一个基于Vue 3、TypeScript和Vue-Flow的流程编排工具项目。

## 项目特性
- 可视化流程编辑器
- 支持多种节点类型（开始、处理、判断、结束）
- 拖拽创建和编辑节点
- 智能连线功能
- 撤销/重做操作
- 数据持久化和导入/导出
- 自动保存功能

## 技术栈
- Vue 3 + Composition API
- TypeScript
- Vue-Flow (可视化流程图库)
- Vite (构建工具)

## 编码规范
- 使用Composition API风格
- 所有组件使用TypeScript
- 遵循Vue 3最佳实践
- 使用响应式数据管理状态
