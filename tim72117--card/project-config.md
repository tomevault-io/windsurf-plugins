# 專案開發規範 (Project Rules)

本專案開發過程與 AI 協作必須嚴格遵守以下規範：

1. **語言限制**：
   - 必須使用 **繁體中文**。
   - 嚴格禁止使用 **簡體中文**。

2. **實作原則**：
   - **不要過度規劃，需要先詢問更具體的細節**。
   - **依照提示詞範圍實作**。
   - **當提示詞過於廣泛或模糊時，不要提供實作方案，先詢問更具體的細節**。
   - **不要進行自動化測試**。
