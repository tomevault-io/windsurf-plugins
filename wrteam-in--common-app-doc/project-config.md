# Site Configuration

This documentation site uses Docusaurus, which provides several configuration options.

## Main Configuration

- [docusaurus.config.js](mdc:docusaurus.config.js): The main configuration file
  - Site metadata (title, tagline, URL)
  - Theme configuration
  - Navigation setup
  - Plugin configuration

## Sidebar Configuration

- [sidebars.js](mdc:sidebars.js): Defines the structure of the documentation sidebar
  - Controls which pages appear in the sidebar
  - Sets the order of pages and categories

## Customizing Appearance

- [src/css/custom.css](mdc:src/css/custom.css): Custom CSS for styling the site
  - Override default styles
  - Define custom color schemes
  - Add custom fonts

## Deployment Configuration

Deployment settings are in [docusaurus.config.js](mdc:docusaurus.config.js):
- `url`: The site's URL
- `baseUrl`: The base URL path
- `organizationName`: Usually your GitHub organization/username
- `projectName`: Usually your repository name
