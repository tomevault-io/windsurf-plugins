# 김비서 — 업무 대시보드

마케팅팀의 일일 업무, 일정, 매출, 프로젝트 진행 현황을 한눈에 관리하는 대시보드입니다.

## 주요 기능

- **할 일 관리** — 우선순위별 업무 목록, 상태 추적, 마감일 관리
- **주간 일정** — 요일별 미팅 및 마감 일정 한눈에 보기
- **매출 분석** — 제품별/지역별 매출 차트 (도넛, 바 차트)
- **프로젝트 현황** — 6개 진행중 프로젝트의 진행률 및 예산 현황
- **업무 흐름도** — 기획 → 제작 → 검토 → 배포 → 분석 단계 시각화
- **다크/라이트 모드** — 테마 토글 지원

## 파일 구조

```
워크숍-실습/
├── CLAUDE.md                          # 이 파일 (프로젝트 정보)
├── .claude/
│   ├── launch.json                    # Claude Code 설정
│   └── skills/김비서/
│       └── SKILL.md                   # /김비서 슬래시 명령어 정의
├── 김비서-데이터/
│   ├── dashboard.html                 # 메인 대시보드 (HTML + CSS + JS)
│   ├── 매출데이터.csv                 # 1월~2월 매출 데이터
│   ├── 업무목록.csv                   # 진행중인 업무 10건
│   ├── 주간일정.txt                   # 3/10~3/14 일주일 일정
│   ├── 프로젝트현황.csv               # 6개 프로젝트 진행률 및 예산
│   └── 회의록.txt                     # 3/10 마케팅팀 회의 기록
└── 정리해줘/                          # 파일 정리 결과
    ├── 보고서/                        # 보고서_초안/수정/최종/진짜최종
    ├── 메모/                          # 기획, 아이디어, 피드백
    ├── 업무/                          # 할일, 회의실, 영수증
    └── 기타/                          # 임시파일, 나중에정리, 링크모음
```

## 사용 기술

| 기술 | 용도 |
|------|------|
| **HTML5** | 마크업 및 페이지 구조 |
| **CSS3** | 다크/라이트 테마, 반응형 레이아웃, 그리드/플렉스 |
| **JavaScript (Vanilla)** | 인터랙션, 차트 렌더링, 테마 토글, 업무 상태 관리 |
| **Chart.js 4.4.0** | 도넛 차트, 바 차트 |

## 사용 방법

### 1. 대시보드 열기
```bash
# 로컬 브라우저에서 HTML 파일 열기
open 김비서-데이터/dashboard.html
```

### 2. 데이터 갱신 및 브리핑 받기
```
/김비서
```
자동으로 모든 데이터 파일을 읽고, 대시보드를 최신화하고, 오늘의 브리핑을 제공합니다.

### 3. 테마 전환
우측 상단 **🌙 다크 모드 / ☀️ 라이트 모드** 버튼 클릭

## 데이터 업데이트

CSV/TXT 파일을 수정하면 `/김비서` 명령어로 대시보드에 자동 반영됩니다.

- `매출데이터.csv` — 제품명, 수량, 단가, 매출액, 지역 추가
- `업무목록.csv` — 업무, 우선순위, 상태, 담당자, 마감일 입력
- `주간일정.txt` — 일정 업데이트
- `프로젝트현황.csv` — 진행률, 예산 갱신
