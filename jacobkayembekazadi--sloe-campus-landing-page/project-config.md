# Project Configuration & Standards

## Tech Stack
- **Framework:** React 19 (Functional Components + Hooks)
- **Build Tool:** Vite 6
- **Language:** TypeScript 5.8 (Strict Mode)
- **Styling:** Tailwind CSS 3 (Mobile-first utility classes)
- **Package Manager:** npm

## Coding Conventions
- **Components:** Use PascalCase filenames in `components/`. Co-locate types/interfaces with the component.
- **TypeScript:** Avoid `any`. Define strict interfaces for all props.
- **Styling:** Use Tailwind classes directly in JSX `className`. Avoid inline styles or separate CSS files unless absolutely necessary.
- **Assets:** Reference images from `public/images/` as absolute paths (e.g., `/images/file.png`).

## Project Structure
- `components/`: All UI components (Hero, Pricing, etc.).
- `utils/`: Helper logic (pixel tracking, email capture).
- `public/`: Static assets.

## Key Commands
- Start Dev Server: `npm run dev`
- Build for Production: `npm run build`
