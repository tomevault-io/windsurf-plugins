# プロジェクト基本情報

このプロジェクトは Python で書かれた Web アプリケーションです。
FastAPI と htmx を用いて、簡易的な SPA を作成することを目的にします。

# 技術スタック

- API: FastAPI
- フロントエンド: htmx
- CSS: Tailwind CSS, daisy UI
- python パッケージ管理: uv

# コマンド

## Python のコードフォーマットとタイプチェック

Python のコードフォーマット及びタイプチェックは下記のコマンドを利用してください。

```bash
uv run poe check
```
