# Your rule content
1. Use TailwindCSS v4 for styling, you can find the documentation [here](mdc:https:/tailwindcss.com/docs)
2. New components should be created in the directory src>components
3. Style for this project is neo brutalism, elements should have a black outline 3px minimum, when I ask you to add a shadow, you should add a pure black rectangle same size as the element behind the element offset by Hpx to the left and Hpx to the bottom, this will mimick the shadow. For hover effect, do translate-x-[-Hpx] hover:translate-y-[Hpx] on the element so that it hides the shadow. H is a value that can be equal to 4 for example.
