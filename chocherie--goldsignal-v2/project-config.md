# Read the Spec Before Coding

Before making changes to Python scripts or specs, read the spec for the system you're touching.

## How to Find the Right Spec

1. Check `specs/README.md` for the index.
2. Data merge / CSV outputs → `specs/data-pipeline.md`
3. Bloomberg paste layout → `specs/bloomberg-bdh-paste-plan.md`, `references/bloomberg-data-guide.md`

## If the Spec and Code Disagree

1. Determine which is correct (ask the user if unclear).
2. Update whichever is wrong.
3. Never silently diverge.
