# Utility Functions

- Create reusable service independant utility functions in `internal/utils/` directory with relevant file name based on the thing they work on.
- Place all reusable functions that has logic specific to this project in `internal/shared/helpers` with relevant file name based on the thing they work on.
- Before creating any utlity or helper function, scan the respective directory to check whether similar function exists which can be reused.
