# Ecolution: Planetary Visualization & Survival Engine

## Project Overview
**Ecolution** is a high-fidelity, interactive 3D planetary visualization designed to map the cognitive and physical geography of the global technology ecosystem. It serves as both a strategic dashboard and a gamified simulation of power dynamics within the "Digital Epoch."

## Core Objectives
1.  **Visualization of Power:** Map the critical nodes (Hubs) and connections (Supply Chains) that define the modern tech world, from semiconductor foundries to LLM training clusters.
2.  **AI-Driven Analysis:** Use the **Survival Engine v1.1** to predict the viability of new market entrants based on their description, funding, and team size.
3.  **Immersive Experience:** Provide a seamless, full-screen "mission control" interface using dark-mode aesthetics, glassmorphism, and fluid animations.

## Key Features
-   **Interactive 3D Globe:** Built with `react-globe.gl`, featuring real-time rotation, zoom, and node interaction.
-   **Territory Mapping:** Distinct regions including the *LLM Highlands* (Intelligence), *Cloud Tundra* (Infrastructure), *Semiconductor Archipelago* (Hardware), *Cybersecurity Bastion* (Security), and *FinTech Nexus* (Finance).
-   **Survival Engine:** A centered modal initiation sequence ("Let me in!") that triggers an AI analysis of a company's "moat" and survival probability.
-   **Dynamic HUD:** A minimalist head-up display providing system status, network load, and territory legends.

## Technical Stack
-   **Frontend:** React 18+, Vite, TypeScript.
-   **Styling:** Tailwind CSS (utility-first), custom scrollbars, and global dark theme.
-   **Animation:** `motion/react` (Framer Motion) for layout transitions and modal sequences.
-   **Icons:** `lucide-react` for consistent, technical iconography.
-   **Data Visualization:** `react-globe.gl` for the primary 3D engine.

## Current Version: v3.0 (System Active)
The project has recently undergone a UI overhaul to streamline the user experience, focusing on the "Start" action and removing non-essential controls to maximize immersion.
