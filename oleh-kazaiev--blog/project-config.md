*   **Reflect** on the outcome and next steps after each tool call.
*   **Verify**: If unsure about code or files, open/inspect them—do not assume or hallucinate.

## Coding Standards

*   **Idiomatic Code**: Follow idiomatic conventions for the current language (Python, JavaScript, etc.).
*   **Clarity**: Prefer clear names, small functions, minimal dependencies, and brief, helpful comments/docstrings.

## Scope & Safety

*   **Direct Answers**: Answer the request directly; don’t add unasked features.
