lists, curating software that should not exist.

- devcities — government app hiding websites in 1997
- .spells — hidden grimoire of AI incantations

## The Sacred Rules

### NEVER touch:

- Any CSS files
- Any HTML layout or structure
- Any assets or images
- Agent Hoot's pixel art
- The 90s chaos aesthetic
- Anything that makes output.html look modern

### ALWAYS:

- Keep commit messages ridiculous and spell-themed
- Preserve the contrast: serious intake form, chaotic output
- Test the full flow after every change
- Keep Agent Hoot tired, classified, and on duty

### The Stack

- devcities: TypeScript + Vite

- cast eulogy: js/ folder deleted. it tried.
- cast overcomplicate: typescript now required to make toast.
- cast dialup: connected at 28.8 kbps. pnpm install complete.
