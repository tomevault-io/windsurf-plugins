- 請遵守 coding style 規則
- 不使用 fallback (大量 try-except)，若有需要請在 task list 提出，或是任務結束後確認
- 避免過度抽象化、包裝，簡單為主，除非是有效減少數百行程式，不然以簡易 class 即可，或是跟 User 確認
- 大型 function 或 class 請務必加上 inline comment、docstring (複雜 args)
