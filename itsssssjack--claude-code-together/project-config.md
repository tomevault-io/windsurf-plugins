# 🏛️ Apex Studio — Project Constitution

## Identity
- **Project:** Apex Studio — Web Design Agency Landing Page
- **Stack:** React + Vite + Tailwind CSS v3
- **Type:** Static single-page website (no backend, no routing)

## Data Schema
No dynamic data. All content is hardcoded as static copy.

### Page Sections (ordered)
1. **Navbar** — Logo, nav links, CTA button
2. **Hero** — Headline, description, CTA, browser mockup
3. **Social Proof** — "Trusted by" + 6 logos
4. **Features** — 6 benefit cards (icon, title, description)
5. **Testimonials** — 3 customer quotes (name, title, photo)
6. **Contact Form** — Name, email, message fields + submit
7. **Call to Action** — Repeated CTA before footer
8. **Footer** — Logo, nav columns, social icons, copyright

## Design System
- **Palette:** Blue primary (#3B82F6), Gold accent (#D97706), neutral grays
- **Vibe:** Clean, modern, premium SaaS (Linear/Notion inspired)
- **Typography:** Inter (Google Fonts)
- **Spacing:** Generous whitespace, rounded-xl corners, soft shadows
- **Animations:** Intersection Observer fade-in on scroll, hover states

## Behavioral Rules
- Mobile-first, fully responsive
- CTA appears above fold and repeated near bottom
- Real, realistic copy — no lorem ipsum
- Production-ready, clean code
