# Computer Services Website Project

This project is a computer services website with admin panel for managing articles about computer services and repairs.

## Technologies
- Next.js 14 with TypeScript
- Tailwind CSS for styling
- App Router architecture
- JSON file storage for articles
- Responsive design
- Uzbek language interface

## Features
- Article display with categories
- Search functionality
- Admin panel for CRUD operations
- Modern responsive design
- User-friendly interface

## Project Structure
```
src/
├── app/
│   ├── globals.css          # Global CSS styles
│   ├── layout.tsx           # Root layout
│   ├── page.tsx             # Home page
│   ├── maqolalar/
│   │   └── page.tsx         # Articles page
│   ├── admin/
│   │   └── page.tsx         # Admin panel
│   └── api/
│       └── articles/
│           └── route.ts     # API endpoint
└── data/
    └── articles.json        # Articles data
```

## Available Scripts
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint

## URLs
- Home: http://localhost:3000
- Articles: http://localhost:3000/maqolalar
- Admin Panel: http://localhost:3000/admin
