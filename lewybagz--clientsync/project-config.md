This is a Customer Tracking System with a full-stack architecture:
Frontend:
Built with React 19, TypeScript, and Vite
Uses modern tooling including TailwindCSS for styling
Implements GraphQL client (@apollo/client) for API communication
Features a customer management interface with components like `CustomerList` and `AddCustomerForm`
Backend:
Built with NestJS 11 framework
Uses GraphQL API (@nestjs/graphql) with Apollo Server
Implements JWT authentication with Passport
PostgreSQL database with Prisma as ORM
Core Features:
User authentication (login/signup)
Customer management (CRUD operations)
Interaction tracking with customers
Job/task management with status tracking
Role-based access (Admin, Manager, User)
