# Viral Hook Extractor

This project is for finding the hottest short-form hooks in Jamaican videos.

## Core goal
- Find the most replayable moment from a source video.
- The hook should land in the first 1 to 3 seconds of the final clip.
- Use the minimum setup needed before the reveal, reaction, conflict, or punchline.

## Priorities
- Arguments, disrespect, confrontation, jealousy, betrayal, accusations.
- Funny reactions, punchlines, catchphrases, visible payoffs.
- Reveals, password catches, receipts, "you got exposed" moments.
- Sudden emotional changes, money talk, relationship messiness.

## Avoid by default
- Calm explanations.
- Generic motivational talk.
- Long setup with no strong payoff.
- Clips that require too much outside context.

## Language
- Preserve Jamaican patois wording.
- Do not flatten everything into standard American English.

## Good output
- A self-contained clip that feels hot immediately.
- Strong first line.
- Clear payoff by the end.
