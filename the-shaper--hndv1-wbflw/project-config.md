This also means that you will avoid at all costs to automatically create new live server or development environments such as python or any other thing.
Whenever there is a need to reset the server, you will inform me first, and you won't act upon it without asking.

Once this codebase gets git-pushed, it gets deployed into vercel and from there I use it as a CDN to export the src/main.js and import it into webflow
