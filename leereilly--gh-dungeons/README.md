# gh-dungeons

> Source: [leereilly/gh-dungeons](https://github.com/leereilly/gh-dungeons) — distributed by [TomeVault](https://tomevault.io)

A GitHub CLI extension roguelike dungeon crawler that turns your codebase into a playable game. Procedurally generated, deterministic dungeons are unique to each repository and SHA.

## Windsurf Config

The `project-config.md` file in this directory is the project config converted for Windsurf.
Original source: `copilot-instructions.md` in [leereilly/gh-dungeons](https://github.com/leereilly/gh-dungeons).
