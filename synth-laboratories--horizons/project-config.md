# Horizons Agent Rules

These rules apply to any work under `Horizons/`.

## No Specs Or Tests In Horizons

- Do **not** add specifications or planning documents under `Horizons/`.
  - Specs live under `specifications/`.
- Do **not** add tests of any kind under `Horizons/`.
  - All testing (unit/integration/property/E2E) lives under `testing/`.
