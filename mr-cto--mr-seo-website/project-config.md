# Mr SEO Website - Next.js 15 Project

## Project Overview

Professional SEO agency website built with Next.js 15, TypeScript, and Tailwind CSS. Features 6 enterprise portfolio case studies and 10 SEO blog posts managed via MDX.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (animations)
- MDX (content management)
- React Hook Form + Zod (form validation)
- next-seo (SEO optimization)
- next-sitemap (sitemap generation)

## Project Structure

- `/app` - Next.js App Router pages and API routes
- `/components` - Reusable UI components
- `/content/portfolio` - MDX case studies for 6 clients
- `/content/blog` - 10 SEO blog posts in MDX format
- `/lib` - Utilities and helpers
- `/public` - Static assets
