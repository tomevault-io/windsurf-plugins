## Git Commit Messages

git commitのメッセージを生成する時は次のフォーマットに従います。

<Type>: <Message>

この時、`<Type>`は以下のような動詞の英単語にします。

- Add: 新機能の追加
- Fix: バグ修正
- Edit: 編集
- Remove: 削除
- Refactor: リファクタリング

`<Message>`部分は、**必ず日本語**にし、簡潔に短い文言にまとめます。
