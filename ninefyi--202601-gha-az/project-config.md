# Project Overview

This project contains training materials and presentations, primarily focused on GitHub and related technologies. The content is organized into folders based on topics.

## Directory Overview

* `gitHub_actions/`: This folder contains materials related to GitHub Actions.
  * `github_actions_crash_course_2026.md`: A 100-slide presentation providing a comprehensive crash course on GitHub Actions for students, including labs and workshops.
* `azure_fundamentals/`: This folder contains materials related to Azure Fundamentals.
  * `azure_fundamentals_course_Jan_2026.md`: A 100-slide presentation providing a crash course on Azure Fundamentals for students, including labs and workshops.

## Usage

The markdown files in this project are designed to be used as presentations. You can use a tool like [Marp](https://marp.app/) to view them as slideshows.

### Running Slideshows with Marp

1. **Install the Marp for VS Code extension.**
2. **Open one of the markdown files** (e.g., `github_actions/github_actions_crash_course_Jan_2026.md`).
3. **Click the Marp icon** in the top-right corner of the editor to preview the slideshow.
4. You can export the slideshow to HTML or PDF by opening the command palette (`Ctrl+Shift+P` or `Cmd+Shift+P`) and typing "Marp: Export".

### Using with GitHub Codespaces

This project includes a `.devcontainer` configuration for GitHub Codespaces. This setup automatically installs `marp-cli` and recommends the Marp for VS Code extension, allowing you to work with the presentations directly in a cloud-based development environment.
