# Copilot Instructions

## Project: Portfolio Website - Full Stack Developer

- Framework: Next.js (App Router, TypeScript, Tailwind CSS)
- Theme: Dark Mode (bg: #0a0f1e)
- Sections: Hero, Skills, Projects, Contact, Footer
- Target: Full Stack Developer portfolio for recruiters

## File Structure
- `app/page.tsx` — main page composing all sections
- `app/layout.tsx` — root layout with metadata
- `app/globals.css` — global dark theme styles
- `components/Navbar.tsx` — sticky responsive navigation
- `components/Hero.tsx` — hero with name, role, CTA, social links
- `components/Skills.tsx` — skill bars + tech badge grid
- `components/Projects.tsx` — project cards with tags & links
- `components/Contact.tsx` — contact form + info
- `components/Footer.tsx` — simple footer

## Placeholders to Customize
- Replace "Your Name" with actual name
- Update GitHub/LinkedIn URLs
- Update email address
- Replace project data in Projects.tsx with real projects
- Update skill levels in Skills.tsx

## Dev Server
Run: `npm run dev` → http://localhost:3000
