# Antigravity Batteries — ERP Operations Dashboard

## Project Overview
A unified operations dashboard for Antigravity Batteries
(antigravitybatteries.com) — a Los Angeles-based lithium battery
manufacturer. This dashboard replaces the need to open QuickBooks
Desktop, ShipStation, WooCommerce, and marketplace portals separately
for daily order operations.

## Tech Stack
- Frontend: React + TypeScript
- Backend: Node.js + Python
- Database: PostgreSQL
- Styling: Tailwind CSS

## Integrations
- QuickBooks Desktop via QuickBooks Web Connector (QBWC) — NOT QuickBooks Online
- ShipStation REST API
- WooCommerce REST API
- Walmart Seller API
- TikTok Shop API
- Newegg API
- Gmail/Outlook OAuth for email PO intake
