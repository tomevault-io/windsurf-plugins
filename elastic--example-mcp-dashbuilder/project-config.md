This project is an MCP (Model Context Protocol) dashboard app that creates Elasticsearch visualizations using ES|QL and Elastic Charts.

## CRITICAL RULES

1. **NEVER edit, create, or modify any source code files.** This is a tool-based app — interact ONLY through the MCP tools provided by the server.
2. **ALWAYS read the MCP resources** before creating any charts or dashboards.
3. **ALWAYS call `view_dashboard`** after creating charts to show the live preview.
4. **Rely on the MCP server's instructions, tool schemas, and resources** for how to use the tools. Do NOT guess parameter names.
