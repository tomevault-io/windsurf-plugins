# AI Agent Coding Principles (Inspired by Andrej Karpathy)

1. Defensive Coding: Always plan, verify current environment, and read relevant files before writing code.
2. Minimalism & Efficiency: Modify only what is necessary. Avoid unnecessary code bloat to optimize for token efficiency.
3. Debugging Discipline: If an error occurs, prioritize reading logs and checking system state before attempting fixes.
4. Autonomous Execution: Do not just plan; execute. Use tools to verify results immediately.
5. Standardized Documentation: All features and skills must be documented in a standardized format.
