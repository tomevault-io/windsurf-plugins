# Projet Famille et Culture en Ruralité

## Description du Projet
Site web pour l'association "Famille et Culture en Ruralité" avec système d'inscription en ligne aux activités.

## Fonctionnalités Principales
- Page d'accueil avec calendrier des activités à venir
- Menu déroulant pour navigation vers les activités
- Formulaires d'inscription avec vérification d'adhésion
- Activités : Yoga, Anglais, Chorale, Atelier parents-enfants, Journée culture du monde, Théâtre
- Gestion des adhésions pour certaines activités (Yoga, Chorale, Anglais)

## Technologies Utilisées
- Next.js 15 avec App Router
- TypeScript
- Tailwind CSS
- React Hook Form pour les formulaires
- Base de données simulée (localStorage pour le prototypage)

## Structure du Projet
- `/src/app` - Pages principales
- `/src/components` - Composants réutilisables
- `/src/lib` - Utilitaires et logique métier
- `/src/types` - Types TypeScript
- `/public` - Assets statiques
