# GoProxy

> Source: [isboyjc/GoProxy](https://github.com/isboyjc/GoProxy) — distributed by [TomeVault](https://tomevault.io)

GoProxy 从多个公开代理源自动抓取 HTTP/SOCKS5 代理，通过严格验证（出口 IP + 位置 + 延迟）后加入智能代理池，对外提供 HTTP 和 SOCKS5 双协议代理服务。系统采用质量分级、智能补充、自动优化等机制，确保代理池始终保持高质量和稳定性。
