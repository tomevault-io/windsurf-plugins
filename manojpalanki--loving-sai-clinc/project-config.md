# Loving Sai Clinic - Project Overview

## Project Description
"Loving Sai Clinic" is a premium, high-conversion medical website built for Dr. T. Sameer Nandan in Visakhapatnam. The project focuses on building trust, demonstrating medical authority, and maximizing patient conversions through a state-of-the-art UI/UX design.

## Tech Stack
* **Frontend Framework:** React (Vite) with TypeScript
* **Styling:** Tailwind CSS (Architectural Sanctuary Design System)
* **Routing:** React Router DOM
* **Icons:** Lucide React
* **Animations:** Framer Motion

## Key Features & Optimizations
* **Premium Medical UI:** Glassmorphism, deep layered shadows (`shadow-[0_20px_60px_rgba(0,0,0,0.08)]`), and strict adherence to a professional `#004ac6` primary color scheme.
* **Omnichannel Conversion:** Global sticky mobile bottom bar, floating WhatsApp integration, and highly visible primary CTAs.
* **Maximum Performance (<2s Load):**
  - React Lazy Loading and Route-level code splitting via `Suspense`.
  - Deferred loading (`loading="lazy"`) for heavy iframes and deep images.
  - Native AVIF image compression.
  - Asynchronous typography rendering (`font-display: swap`).
* **Trust Elements:** Real patient verification badges, 5.0 Google Review indicators, and transparent medical stats natively integrated.

## Core Pages
1. `/` - **Home Page:** High-impact Hero, Services grid, Testimonials, and Final CTA.
2. `/about` - **About Us:** Doctor profile, clinic mission, and specialized care details.
3. `/services` - **Services:** Comprehensive medical services, categorized by specialty (Diabetes, General Medicine, Thyroid, etc).
4. `/contact` - **Contact:** Callback request forms, embedded Google Maps, direct contact routes.

*Built for maximum velocity, accessibility, and client acquisition.*
