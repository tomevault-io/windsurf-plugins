Core Features:
HTML Rendering using code.
HTMX for client-side interactions and partial page updates.
SSE for server-sent events.
Components like Buttons,Labels,Inputs,Selects,TextAreas,        Forms, Tables, datagrids, Charts, etc.

Each component should have a unique identifier.

Testing

Deployment should be done using Docker.
