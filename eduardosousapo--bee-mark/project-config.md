# Projeto
Este repositório constrói o site público da **Beemark**.

## Objetivo do site
Apresentar a Beemark como uma operação que integra:
- posicionamento;
- branding;
- landing pages;
- tráfego pago;
- criativos;
- IA para qualificação;
- CRM e dados;
- passagem do lead para o vendedor humano.

## Identidade desejada
- dark premium
- high-tech
- sofisticada
- estratégica
- orientada à conversão
- moderna sem parecer genérica

## Regras centrais
- no máximo 2 páginas no MVP;
- foco em clareza comercial;
- animação com propósito;
- não transformar o site em um playground visual;
- não parecer agência genérica.
