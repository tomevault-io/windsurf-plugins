## Project Overview
This is a Live Email Designer project built with:
- Vite + React + TypeScript
- Ant Design for UI components
- Monaco Editor for code editing
- Ant Design Splitter for resizable panels
- JS Beautify for HTML formatting

## Code Standards
- All comments must be in English
- Use TypeScript with proper type definitions
- Follow React best practices and hooks
- Use Ant Design components for consistent UI
- Implement responsive design patterns

## Architecture
- Two-panel layout: left (editor) and right (preview)
- Resizable panels using Ant Design Splitter
- Real-time synchronization between editor and preview
- Device preview modes (desktop, tablet, phone)
- HTML wrapper functionality for email template structure
- Code highlighting and auto-completion for HTML
- Proper HTML formatting and indentation
