# Rules

- Don't use classes like `shadow-*` that are not supported in nativewind.
- Use semantic classes like `bg-primary`, `border-primary`, `text-primary`, etc, to maintain the color consistency.
- For backend (supabase), we are using dynamic.xyz auth, not the supabase, so all the RLS and security is handled by the application logic.
