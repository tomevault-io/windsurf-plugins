# HTML Coding Standards

## Structure Requirements
- Use HTML5 semantic elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`)
- Include proper DOCTYPE declaration
- Add viewport meta tag for responsive design
- Use proper heading hierarchy (h1 → h2 → h3)

## Tailwind CSS Integration
- Include Tailwind CSS via CDN in the `<head>`
- Use Tailwind utility classes for styling
- Avoid inline styles - use Tailwind classes
- Follow mobile-first responsive design principles

## Accessibility
- Include `alt` attributes for all images
- Use proper ARIA labels where needed
- Ensure keyboard navigation works
- Maintain proper color contrast

## Performance
- Optimize images for web (WebP format preferred)
- Minimize external dependencies
- Use lazy loading for images when appropriate
