# Robot Keyframe Editor - Copilot Instructions

This is a web-based robot keyframe editor tool with 3D visualization.

## Project Features
- Load URDF files with mesh references
- Load CSV trajectory data (base pose + joint positions)
- Timeline scrubbing for robot state visualization
- Joint angle control panel with sliders and editable values
- Keyframe editing with residual-based system
- Export combined trajectories

## Tech Stack
- Vite for build tooling
- Three.js for 3D visualization
- urdf-loader for URDF file parsing
- Vanilla JavaScript/TypeScript
