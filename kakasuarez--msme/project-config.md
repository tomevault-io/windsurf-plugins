# Copilot Instructions for MSME Digital Platform

## Project Overview

This is a comprehensive MSME (Micro, Small, and Medium Enterprises) Digital Platform prototype built with Next.js, TypeScript, and Tailwind CSS. The platform aims to empower MSMEs through digital transformation, financial inclusion, and business growth opportunities.

## Key Features

- **Invoice Discounting**: Digital invoice management and financing
- **Buyer-Supplier Marketplace**: Connect MSMEs with buyers and suppliers
- **Tender Alerts**: Government and private tender notifications
- **Roadshow Management**: Track and manage outreach activities
- **Business Tools Integration**: CRM, ERP, and accounting tools
- **WhatsApp Business Integration**: Communication and onboarding
- **Analytics Dashboard**: Track platform metrics and MSME performance
- **Green Accelerator Credits**: Sustainability incentives

## Technology Stack

- **Frontend**: Next.js 15 with App Router, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: JSON-based storage for prototype (easily upgradeable)
- **UI Components**: Custom components with Tailwind CSS
- **Icons**: Lucide React
- **Charts**: Recharts for analytics
- **Forms**: React Hook Form with validation

## Architecture Guidelines

- Use App Router pattern for all routes
- Implement server components where possible
- Follow TypeScript best practices
- Use Tailwind CSS for styling
- Implement responsive design
- Follow accessibility guidelines

## Code Style

- Use functional components with hooks
- Implement proper error handling
- Use TypeScript interfaces for data models
- Follow Next.js best practices for performance
- Implement proper SEO optimization
