## Project Context
This is a professional portfolio website built with Next.js 15, TypeScript, and Tailwind CSS. The portfolio showcases projects, skills, and professional experience.

## Code Style and Best Practices
- Use TypeScript for all components and pages
- Follow React functional component patterns with hooks
- Use Tailwind CSS for styling with consistent design system
- Implement responsive design principles (mobile-first approach)
- Use semantic HTML elements for accessibility
- Follow Next.js App Router conventions
- Optimize for SEO with proper meta tags and structured data
- Ensure fast loading times and Core Web Vitals compliance

## Component Structure
- Keep components modular and reusable
- Use proper TypeScript interfaces for props
- Implement proper error boundaries
- Use Next.js dynamic imports for code splitting when appropriate
- Follow the atomic design principle (atoms, molecules, organisms)

## Content Guidelines
- Focus on professional presentation without revealing proprietary code
- Create detailed case studies for each project
- Include technical explanations and problem-solving approaches
- Showcase skills through interactive demos and code examples
- Maintain professional tone throughout the content

## Performance and SEO
- Use Next.js Image component for optimized images
- Implement proper meta tags for each page
- Use structured data for better search visibility
- Optimize loading states and animations
- Ensure accessibility compliance (WCAG guidelines)
