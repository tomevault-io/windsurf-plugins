# Tab to File Mapping

Each tab in the dashboard UI is controlled by a specific JavaScript file. This file is responsible for all the logic, chart creation, and event handling within that tab.

When you need to modify a chart or functionality on a specific tab, you should start by looking at its corresponding file.
