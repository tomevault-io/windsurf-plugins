# Interface Naming Convention

所有接口必须以 `I` 开头，例如：

- ✅ `ILogManagerContract`
- ✅ `ILogChannelContract`
- ✅ `ILogDriverContract`
- ❌ `LogManagerContract`
- ❌ `LogChannelContract`
- ❌ `LogDriverContract`

这个规则适用于整个项目中的所有接口定义。
