google("gemini-pro-latest")    // ✅ Para análises complexas

### 📊 Quando Usar Cada Modelo

| Modelo | Use Para | Velocidade | Custo |
|--------|----------|------------|-------|
| `gemini-pro-latest` | Análises complexas, código, raciocínio | Médio | Alto |
| `gemini-flash-latest` | Chat, respostas rápidas, FAQs | Rápido | Baixo |

## 🚨 Problemas Comuns Resolvidos

### 1. API Key Inválida
**Erro**: "API key not valid"

**Solução**: 
```bash
# Verificar .env.local
cat .env.local
# Deve ter: GOOGLE_GENERATIVE_AI_API_KEY=AIzaSy...

# Reiniciar servidor
# Ctrl+C e depois: pnpm dev
```

**Guia**: GUIA_CONFIGURACAO_GEMINI.md#api-key-não-funciona

---

### 2. Modelo Não Encontrado
**Erro**: "Model not found" ou 404

**Solução**:
```typescript
// ❌ ERRADO
google("gemini-1.5-pro")

// ✅ CORRETO
google("gemini-pro-latest")
```

**Guia**: GUIA_CONFIGURACAO_GEMINI.md#modelo-não-encontrado

---

### 3. Streaming Não Funciona
**Erro**: Resposta não aparece em tempo real

**Checklist**:
- [ ] API route retorna `.toDataStreamResponse()`?
- [ ] Cliente usa `useChat()` do `ai/react`?
- [ ] Sem cache na route?

**Guia**: GUIA_CONFIGURACAO_GEMINI.md#streaming-não-funciona

---

### 4. Erro 429 (Rate Limit)
**Erro**: "Too Many Requests"

**Soluções**:
1. Aguardar alguns minutos
2. Usar `gemini-flash-latest` (mais leve)
3. Verificar quota no Google AI Studio

**Guia**: GUIA_CONFIGURACAO_GEMINI.md#erro-429

## 🎓 Conceitos Importantes

### 1. Streaming
Respostas aparecem token por token em tempo real (como ChatGPT)

```typescript
// Backend
return result.toDataStreamResponse();

// Frontend
const { messages } = useChat();
```

### 2. System Prompts
Define comportamento da IA

```typescript
system: "Você é um assistente prestativo..."
```

### 3. Temperature
Controla criatividade (0.0 = preciso, 1.0 = criativo)

```typescript
temperature: 0.7  // Balanceado
```

### 4. Tools
Funções que a IA pode chamar

```typescript
tools: {
  getWeather: { ... }
}
```
