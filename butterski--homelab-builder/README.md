# homelab-builder

> Source: [Butterski/homelab-builder](https://github.com/Butterski/homelab-builder) — distributed by [TomeVault](https://tomevault.io)

HLBuilder is a interactive web application designed to simplify the process of planning and architecting home laboratory infrastructure. It provides users with a visual interface to design network topologies, receive intelligent hardware recommendations based on their self-hosting needs, and generate actionable shopping lists.
