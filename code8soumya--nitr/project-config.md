## 🏗️ Project Overview & Architecture
NITR HUB is a campus social application for NIT Rourkela students. The app operates through independent modules:
1. **Tab 1: Social Media (Hype Feed)** - Posts, hypes (likes), comments, hashtags.
2. **Tab 2: Campus Utilities (Q&A + Lost & Found)** - Combined feature with gender-filtered sub-tabs.
3. **Tab 3: Daily Quest** - 1-on-1 AI-matched peer chat (30 messages threshold).

### Tech Stack
- **Frontend**: React Native, Expo Router, NativeWind (Tailwind), Zustand.
- **Backend / DB**: AWS Lambda, API Gateway, Aurora PostgreSQL.
- **Auth & Storage**: AWS Cognito, SES, S3, CloudFront CDN.

### Core Principles
* **Module Isolation**: Each module/tab (`src/modules/[name]/`) is heavily isolated with its own Zustand store and wrapped in a boundary.
* **Database Access**: All repository access goes through wrapper functions using transactional safety (e.g., `withTransaction`).
* **Android-First UI**: Prioritize Android Material guidelines and React Native component optimizations over purely web-based styling.
