# YouTube Playlist Audio Downloader - Instrucciones

Aplicación web para descargar y escuchar audios de playlists de YouTube localmente.

## Stack Tecnológico
- Backend: Python con Flask
- Frontend: HTML, CSS, JavaScript
- Descarga de videos: yt-dlp
- Base de datos: Sistema de archivos local

## Características principales
- Descarga audios de playlists de YouTube
- Reproductor de audio integrado
- Gestión de archivos (reproducir, eliminar)
- Interfaz web responsiva

## Dependencias
- Flask 2.3.2
- yt-dlp 2023.12.30
- requests 2.31.0
