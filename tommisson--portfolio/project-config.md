## Directory Overview
This repository contains the source code for a high-performance, "Enterprise Grade" digital portfolio built with Angular and Analog.

The ultimate goal is a technical showcase that is automated, scalable, and near-zero maintenance.

## Intended Tech Stack
- **Frontend Framework**: [Analog](https://analogjs.org/) (Angular meta-framework) for SSR/SSG.
- **State Management**: Angular Signals.
- **Styling**: Tailwind CSS 4.0.
- **Animation**: GSAP + ScrollTrigger.
- **Data Integration**: GitHub API (for project sync).
