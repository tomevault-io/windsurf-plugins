## Vue d'ensemble du projet
Ce projet est une application React moderne utilisant Vite comme outil de build. Il utilise JavaScript (pas TypeScript) et est configuré avec ESLint pour la qualité du code.

## Conventions de code
- Utilisez JavaScript (JSX) pour tous les composants React
- Suivez les conventions de nommage camelCase pour les variables et fonctions
- Utilisez PascalCase pour les noms de composants React
- Préférez les composants fonctionnels avec les hooks React
- Utilisez des noms de fichiers descriptifs avec l'extension .jsx pour les composants

## Structure du projet
- `src/` : Code source principal
- `public/` : Assets statiques
- `src/main.jsx` : Point d'entrée de l'application
- `src/App.jsx` : Composant principal de l'application

## Outils et dépendances
- **React 19.1.0** : Bibliothèque UI principale
- **Vite** : Outil de build rapide et serveur de développement
- **ESLint** : Linting et qualité du code
