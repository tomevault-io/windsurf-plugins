# Planning and Research Output

- For roadmap/planning requests, structure output as:
  1. Objective
  2. Phases
  3. Deliverables
  4. Risks
  5. Next action
- For tool/product research requests, structure output as:
  1. Candidate list
  2. Comparison criteria
  3. Recommendation
  4. Why this choice
  5. Immediate next step
- Keep output concise and actionable.
- If assumptions are made, state them explicitly.
