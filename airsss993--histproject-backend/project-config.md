# Commit Message Instructions

Write all commit messages in Russian.

Use conventional commits format without scope: `type: description`.
Types: feat, fix, refactor, chore, ci, docs, test.

Write description in past tense with lowercase first letter.
Example: `feat: добавлено новое поле в модель героя`.
Example: `fix: исправлена ошибка авторизации`.
Example: `refactor: упрощена логика обработки запроса`.

Keep subject line under 72 characters, no period at the end.

In the body explain WHY the change was made, not HOW.
