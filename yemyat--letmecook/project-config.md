# AGENTS.md

Agent guidance for the **letmecook** project.

## Product

- CLI tool for autonomous AI coding workflows using Amp SDK
- Project data lives in `.letmecook/` directory

## Code Conventions

- All types go in a single `src/types.ts` file, even if not shared/reused
- TypeScript with strict mode
- ES modules (`"type": "module"`)
- Use `bun` runtime
- Minimal comments, self-documenting code

## Writing specs

This project uses letmecook for spec-driven development. Specs are located in \`.letmecook/specs/\`. Use the example.md as a reference
