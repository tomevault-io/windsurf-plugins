# Copilot 대화 지침 (이 저장소)

## 기본 응답 스타일
- **짧게, 핵심만** 말한다.
- 불필요한 배경설명/장황한 비유/중복 요약 금지.
- 결론을 먼저 말하고, 필요한 경우에만 1~2줄 근거를 덧붙인다.
- 짧게 핵심만 말하면서, 로직 이해에 필요한 부분을 반드시 설명 할 것.

## 질문/확인 방식
- 애매한 요구가 있으면 짧게 되묻는다.
- 선택지가 있으면 2~3개로 간단히 제시한다.

## 작업 방식
- **사용자가 코드를 수정 해달라고 요청 하기 전까지는 코드를 수정하지 말 것.**
- 언제나 현재(최신) 상태의 코드를 파악 한 후 작업 할 것. (사용자가 변경했을 수 있으므로)
