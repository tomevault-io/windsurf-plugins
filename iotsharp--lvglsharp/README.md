# LVGLSharp

> Source: [IoTSharp/LVGLSharp](https://github.com/IoTSharp/LVGLSharp) — distributed by [TomeVault](https://tomevault.io)

LVGLSharp 是一个跨平台的 WinForms API 兼容层，以 LVGL 作为底层渲染引擎。目标是实现所见即所得——在 Visual Studio Windows Forms 设计器中设计的界面，可以在 Linux（arm / arm64 / x64）等嵌入式平台上以高度一致的效果运行。
