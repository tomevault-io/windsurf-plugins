# GloryRole

> Tome by [franklesniak](https://github.com/franklesniak/GloryRole) — distributed by [TomeVault](https://tomevault.io)

An unsupervised role mining engine written in PowerShell. Feed it cloud activity logs and it figures out who does what, groups similar principals via K-Means clustering, and generates least-privilege custom role definitions--shrinking permission sprawl without breaking anything.
