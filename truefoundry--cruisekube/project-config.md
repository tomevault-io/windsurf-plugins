# Repo Conventions

- For Go package directories, use lowercase ASCII names only. If a name needs multiple words, concatenate them with no separators.
- For Go source filenames, prefer a single lowercase word when practical. If multiple words are needed, use continuous lowercase with no separators.
- Reserve underscores in Go filenames only for special Go suffixes such as `_test.go`.
