# MedFund Application Overview

MedFund is a platform for medical fundraising campaigns, designed to help patients raise funds for medical treatments, surgeries, and other healthcare needs. The application allows users to:

1. Create and manage medical fundraising campaigns
2. Browse existing campaigns
3. Make donations to campaigns
4. Track medical records
5. Manage payment information
6. View campaign and payment activity

## Key Pages

- [src/pages/Index.tsx](mdc:src/pages/Index.tsx): Home page
- [src/pages/Dashboard.tsx](mdc:src/pages/Dashboard.tsx): User dashboard
- [src/pages/Campaigns.tsx](mdc:src/pages/Campaigns.tsx): Browse all fundraising campaigns
- [src/pages/CampaignDetail.tsx](mdc:src/pages/CampaignDetail.tsx): View details of a specific campaign
- [src/pages/StartCampaign.tsx](mdc:src/pages/StartCampaign.tsx): Create a new fundraising campaign
- [src/pages/MedicalRecords.tsx](mdc:src/pages/MedicalRecords.tsx): View and manage medical records

## Technology Stack

- Frontend: React, TypeScript, Tailwind CSS, shadcn/ui
- Database: Supabase (transitioning to Drizzle ORM)
- Authentication: Supabase Auth
