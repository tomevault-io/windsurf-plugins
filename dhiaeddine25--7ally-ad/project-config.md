# 7ally_ad E-commerce Project Instructions

## Project Overview

Modern e-commerce website for 7ally_ad dessert brand using Next.js, React, and Tailwind CSS.

## Tech Stack

- Next.js (React framework)
- Tailwind CSS for styling
- React hooks for state management
- Responsive design (mobile + desktop)

## Features

- Home page with hero section
- Shop page with product grid
- Product details page
- Shopping cart
- Checkout with 58 Algerian wilayas
- Search and filter functionality

## Design Guidelines

- Colors: soft pastel (pink, cream, brown)
- Clean, minimal, professional UI
- Smooth animations
- Mobile-first responsive design

- [x] Project structure created

- [x] Styling completed
- [x] Project tested and running

## Running the Project

- Development server: `npm run dev`
- Build for production: `npm run build`
- Start production server: `npm start`
- Access at: http://localhost:3000
