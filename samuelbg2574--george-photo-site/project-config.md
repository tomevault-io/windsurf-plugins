## Design Rules

- Use varied section padding: `--spacing-section-sm`, `--spacing-section-md`, `--spacing-section-lg`, `--spacing-section-xl`
- Break grid symmetry in at least 1-2 sections per page
- Never use "Get Started" more than once — vary CTA microcopy
- Every page needs real content, not placeholder text
- Add micro-interactions (hover states, scroll animations) to at least 3 sections
