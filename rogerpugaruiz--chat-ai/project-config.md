# Instrucciones para Copilot - Proyecto de Chatbot ML

Este es un proyecto de machine learning para un chatbot similar a Intercom. El proyecto está enfocado en:

1. Procesamiento de lenguaje natural (NLP) para entender mensajes de usuarios
2. Clasificación de intenciones para dirigir consultas a respuestas apropiadas
3. Generación de respuestas naturales para preguntas frecuentes
4. Integración con sistemas de chat como Intercom

Al generar código para este proyecto, considerar:
- Técnicas modernas de NLP usando transformers y sentence embeddings
- Arquitectura modular para separar preprocesamiento, entrenamiento y predicción
- Buenas prácticas para entrenamiento de modelos y evaluación de rendimiento
- Manejo apropiado de datos conversacionales y flujos de diálogo
