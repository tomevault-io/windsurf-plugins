# Effect3 AI - Task Tracker PWA

## Project Overview
A Progressive Web App for personal task tracking focused on closing 5 clients in 10 days for Effect3 AI (theeffect3.com).

## Tech Stack
- **Backend**: Node.js + Express
- **Database**: SQLite via better-sqlite3
- **Frontend**: Vanilla HTML/CSS/JS (no framework)
- **Theme**: Black (#0a0a0a) and Orange (#ff6b00) minimalistic
- **PWA**: Service Worker + Web App Manifest

## Features
- Task management with categories (Content, LinkedIn, Reddit, Instagram, Facebook, Lead Gen)
- 10-day countdown with progress tracking
- Google Calendar API integration for reminders
- Apple Reminders integration via AppleScript (macOS)
- Productivity monitoring (tab/app tracking)
- Mobile-responsive dashboard

## Development
- Run `npm start` to start the server
- Access at http://localhost:3000
- Install as PWA on mobile via "Add to Home Screen"
