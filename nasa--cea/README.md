# cea

> Source: [nasa/cea](https://github.com/nasa/cea) — distributed by [TomeVault](https://tomevault.io)

CEA computes the equilibrium composition of mixtures via free-energy minimization, and uses the resulting product concentrations to determine thermodynamic and transport properties. It includes a large database of species with thermodynamic curve-fit data, and can be used for rocket performance, detonation parameters, and shock-wave analysis.
