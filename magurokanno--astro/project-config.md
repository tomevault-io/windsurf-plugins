## 4. ワークフロー別プロンプト集

日常の開発で頻繁に使うコマンドのテンプレートです。

### Git / コミットメッセージ生成

ステージングされた変更内容を元に、Conventional Commits形式のコミットメッセージを生成します。

**コマンド例:**
`git diff --staged | gemini "この差分から、Conventional Commits形式のコミットメッセージを1行で生成して。形式は 'type(scope): subject' で、日本語で記述して。"`

### Astro / コンポーネント生成

新しいAstroコンポーネントの雛形を、指定したパスに直接生成します。

**コマンド例:**
`gemini "src/components/common/Button.astro というファイルを作成して。内容は、基本的なボタンのAstroコンポーネントで、propsとしてlabelとvariantを受け取れるようにTypeScriptで型定義もして。" > src/components/common/Button.astro`

### TypeScript / 型定義の生成

クリップボードにコピーしたJSONデータから、TypeScriptの`interface`を生成します。

**macOS のコマンド例:**
`pbpaste | gemini "このJSONデータ構造から、TypeScriptのinterfaceを生成してください。インターフェース名は'ApiResponse'でお願いします。"`

**Windows (PowerShell) のコマンド例:**
`Get-Clipboard | gemini "このJSONデータ構造から、TypeScriptのinterfaceを生成してください。インターフェース名は'ApiResponse'でお願いします。"`
