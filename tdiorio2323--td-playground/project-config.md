## Project Overview

This repository, **TD Playground**, is a personal development sandbox for building, testing, and refining UI/UX components for TD Studios projects. It is a React/TypeScript project that uses Vite for building and Vitest for testing. It uses `pnpm` as the package manager.

## Building and Running

### Prerequisites

- [pnpm](https://pnpm.io/installation)

### Key Commands

- **`pnpm dev`**: Starts the development server.
- **`pnpm build`**: Builds the project for production.
- **`pnpm test`**: Runs the test suite.
- **`pnpm test:coverage`**: Runs the test suite and generates a coverage report.
- **`pnpm lib`**: Runs the design library CLI.
- **`pnpm routes:preview`**: Generates route previews.

## Development Conventions

- **Styling**: The project uses Tailwind CSS with `shadcn/ui`.
- **State Management**: The project uses TanStack Query for state management.
- **Routing**: The project uses React Router v6 for routing.
- **Testing**: The project uses Vitest and React Testing Library for testing.
- **Path Aliases**: The project uses path aliases for easier imports. The `@` alias is configured to point to the `src` directory.
