# Project-Specific Instructions for GitHub Copilot

This is a Streamlit data visualization dashboard project for analyzing datasets using Python. When providing assistance, please consider the following:

## Project Structure
- `src/app.py` - Main Streamlit application
- `src/data_loader.py` - Functions for loading and preprocessing data
- `src/visualizations/` - Modules for creating different types of visualizations
- `data/` - Directory for dataset files

## Libraries Used
- Streamlit for the web interface
- Pandas and NumPy for data manipulation
- Plotly, Matplotlib, and Seaborn for visualizations
- scikit-learn for basic machine learning functionality

## Coding Guidelines
- Use Streamlit best practices for interactive components
- Implement proper error handling for data operations
- Use Plotly as the primary visualization library for interactivity
- Prioritize readable, maintainable code with appropriate comments
- Follow PEP 8 style guidelines for Python code

## Feature Development
- When suggesting new features, focus on enhancing the interactivity of the dashboard
- Consider user experience and performance when adding functionality
- Provide suggestions that maintain compatibility with the existing visualization modules
- Suggest improvements for better data exploration capabilities

## Specific Domain Knowledge
- The dashboard is designed for exploration of various datasets
- Visualizations should support both time series and cross-sectional data analysis
- Suggested code should accommodate both small and large datasets
- Consider adding features for data preprocessing and cleaning when relevant
