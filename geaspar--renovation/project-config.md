# Gemini Project

This file contains project-specific information and context for the Gemini AI assistant.

## Project Purpose

This project is for planning a full house renovation and garden redesign for a property in Ballyfermot. The primary guide for the required work will be the official house survey. The project is being managed and documented via GitHub, with a focus on documentation rather than code.

## Project Structure

*   **`README.md`**: High-level overview of the project and links to key resources.
*   **`planning.md`**: Detailed month-by-month renovation plan with cost estimates.
*   **`TODO.md`**: A comprehensive to-do list based on the house survey.
*   **`Garden.md`**: High-level permaculture and no-dig garden plan, including the outdoor bath.
*   **`Ducks.md`**: A comprehensive guide to all duck-related systems, including the smart coop, drainable pond, and aquaponics. Contains detailed build guides and IoT information.
*   **`Project_Status.md`**: Tracks the progress of our sessions.
*   **`Pics/`**: Contains images and screenshots for reference.

## Workflow

We are systematically working through the `planning.md` file, adding cost estimates for each phase of the renovation. We will use the `Project_Status.md` file to track our progress and ensure we can easily pick up where we left off in each session.
