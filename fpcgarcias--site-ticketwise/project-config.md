# Proteção de Layout

## ⚠️ REGRA CRÍTICA - JAMAIS ALTERE O LAYOUT SEM AUTORIZAÇÃO

**NUNCA, EM HIPÓTESE ALGUMA, ALTERE O LAYOUT SEM AUTORIZAÇÃO EXPRESSA DO USUÁRIO.**

### Diretrizes Obrigatórias:

1. **Proibição Absoluta**: Qualquer modificação em:
   - Estrutura de componentes de interface
   - Estilos CSS/SCSS/Tailwind
   - Posicionamento de elementos
   - Layout de páginas
   - Design visual
   - Responsividade

2. **Antes de Qualquer Alteração de Layout**:
   - Sempre perguntar explicitamente ao usuário
   - Aguardar autorização clara e específica
   - Nunca assumir que mudanças são bem-vindas

3. **Tipos de Alterações Restritas**:
   - Modificações em arquivos de estilo
   - Alterações em componentes visuais
   - Mudanças de estrutura HTML/JSX
   - Ajustes de responsividade
   - Alterações de cores, fontes, espaçamentos

4. **Exceções Permitidas**:
   - Correções de bugs específicos quando solicitado
   - Melhorias de performance que não afetem a aparência
   - Adição de funcionalidades sem impacto visual

### 🚨 LEMBRETE CONSTANTE
Se há dúvida sobre se uma alteração afeta o layout, **SEMPRE PERGUNTE PRIMEIRO**.

O layout é propriedade exclusiva do usuário e deve ser preservado a todo custo.
