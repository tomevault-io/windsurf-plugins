# TypeScript/React 代码风格

## 基本要求
- 严格使用 TypeScript，避免 `any`
- 注释使用中文
- 禁止 emoji 注释

## 代码位置
| 类型 | 目录 |
|-----|------|
| 组件 | `web/src/components/` |
| Hooks | `web/src/hooks/` |
| API | `web/src/lib/` |
| 类型 | `web/src/types/` |

## 端口
- 前端 (Next.js): `3000`
- 后端 (FastAPI): `8000`
