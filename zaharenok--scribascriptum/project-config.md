# Handwriting to Markdown for Obsidian

## Описание

Проект называется ScribaScriptum.

Плагин для Obsidian, который использует Google Gemini API для распознавания рукописного текста на скриншотах и преобразует его в форматированный Markdown. Плагин особенно хорошо справляется с распознаванием:

- Текстовых заметок
- Маркированных и нумерованных списков
- Математических формул
- Таблиц
- Заголовков разного уровня

Изображения, которые не поддаются распознаванию как текст, сохраняются как вложения в заметку.

## Возможности

- 📝 Распознавание рукописного текста с помощью Google Gemini
- ✍️ Автоматическое форматирование в Markdown
- 📊 Поддержка сложных структур (списки, таблицы, формулы)
- 🖼️ Сохранение нераспознанных элементов как изображений
- 🔐 Безопасное хранение API-ключа Google Gemini
- ⚡ Быстрая обработка скриншотов

## Требования

- [Obsidian](https://obsidian.md/) (последняя версия)
- Аккаунт Google с доступом к Gemini API
- API ключ Google Gemini
- Предусмотреть возможность авторизации и платной подписки с премиум функциями

## Установка

1. Установите плагин через Community plugins в настройках Obsidian
2. Получите API ключ на [Google AI Studio](https://makersuite.google.com/)
3. Введите API ключ в настройках плагина
4. Авторизация и получение API ключа доступна только для платных подписчиков
5. 

## Использование

1. Сделайте скриншот рукописной заметки
2. В Obsidian используйте команду `Handwriting to MD: Convert from clipboard`
3. Дождитесь обработки (зависит от размера скриншота)
4. Получите отформатированный Markdown в вашей заметке

## Бесплатная версия

Бесплатная версия плагина доступна для всех пользователей. Она поддерживает:

- 5 запросов в день
- Поддержка только английского языка
- Только загрузка изображений (JPG, PNG)
- Без поддержки PDF

## Платная версия

Платная версия плагина доступна только для платных подписчиков.

- 50 запросов в день
- Поддержка нескольких языков
- Поддержка кастомного промта
- Поддержка кастомных настроек
- Загрузка и обработка PDF-документов
- Пакетная обработка нескольких страниц PDF
- Экспорт результатов в PDF
