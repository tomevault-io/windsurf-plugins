# Copilot Instructions

When contributing to this repository, always follow these sources of truth:

1. `README.md`
2. `CONTRIBUTING.md`
3. `CODE_OF_CONDUCT.md`

## Required contribution rules

- Only add genuine free and open source projects.
- Keep entries in alphabetical order within each section.
- Use this exact entry format:
  `- [Name](https://github.com/owner/repo) - Short, meaningful description.`
- Add concise, value-focused descriptions.
- Avoid duplicate entries.
- Ensure every project link points to a valid GitHub repository URL.

## Collaboration and behavior

- Follow the Contributor Covenant expectations in `CODE_OF_CONDUCT.md`.
- Use respectful, inclusive language in issues, pull requests, and reviews.

## Pull request quality checks

- Keep changes focused and minimal.
- Ensure CI checks pass before requesting review.
- Do not bypass repository rules defined in docs or workflows.
